01 — what is programming · 什么是编程

What is Programming?

什么是编程？

A program is simply a set of instructions. Just like teaching someone to make a sandwich — one step at a time.

程序就是一组清晰、有序的指令。就像教一个机器人做三明治——你需要一步一步地告诉它该做什么。

计算机本身不会"思考"——它只会执行我们给它的指令。编程就是把我们的想法变成计算机能够理解和执行的语言。

🍳 Teaching a robot to make a sandwich — 教机器人做三明治

STEP 1 🍞

Take two slices of bread

拿两片面包

STEP 2 🥩

Add ham

加火腿

STEP 3 🧀

Add cheese

加奶酪

STEP 4 🥪

Close the sandwich

合上三明治

Programming works exactly the same way — clear, ordered instructions.

编程的工作方式完全相同——清晰、有序的指令。

每一步都必须精确，顺序不能乱。少一步或者顺序错误，结果就会不对——这就是为什么编程需要细心和逻辑思维。

05 — mini challenge · 小挑战

Mini Challenge

小挑战

Can you read the code and figure out what happens?

你能读懂代码并判断结果吗？

Q1

What value is stored in x?

x 中存储了什么值？

python
x = 10

The variable x stores the number 10. The = sign assigns the right-side value to the left-side name.

变量 x 存储数字 10。= 号将右边的值赋给左边的名字。

Q2

How many times will "Hi" be printed?

"Hi" 会被打印多少次？

python
for i in range(3): print("Hi")

range(3) creates the sequence [0, 1, 2], so the loop runs exactly 3 times.

range(3) 创建序列 [0, 1, 2]，所以循环运行 3 次。

Q3

What will be printed?

会打印什么？

python
score = 40 if score >= 50: print("Pass") else: print("Fail")

score is 40, which is NOT >= 50, so the else branch runs and prints "Fail".

score 是 40，不满足 >= 50，所以执行 else 分支，打印 "Fail"。

Python Turtle Graphics

Create Art with Code · 用代码创作艺术

下周我们将学习 Python 海龟绘图（Turtle Graphics）。

通过控制一只虚拟的"海龟"在屏幕上移动，你可以绘制出各种美丽的几何图形。这是学习编程最直观、最有趣的方式之一——你写的每一行代码都会立刻变成屏幕上的图案。

Turtle Graphics lets you draw shapes and patterns by telling a virtual turtle where to move.

海龟绘图通过移动一只虚拟海龟来绘制图形。你只需告诉它"前进多少步"、"转多少度"，它就会在屏幕上留下轨迹，形成各种图案。

⬜Squares正方形 — 最基础的几何图形
⭐Stars星形 — 循环 + 角度的组合
🌀Spirals螺旋形 — 逐步增加步长
🔷Geometric Art几何艺术 — 创意无限

预览代码 — Preview Code

python
import turtle # 画一个正方形 for i in range(4): turtle.forward(100) # 前进 100 步 turtle.left(90) # 向左转 90 度

Next week, you'll write code that draws beautiful art.

下周，你将亲手编写代码，让计算机在屏幕上绘制出属于你的图案。

从一个简单的正方形开始，逐步探索更复杂的几何艺术。编程不只是文字，它也可以是艺术。
